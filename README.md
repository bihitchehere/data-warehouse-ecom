# Ecommerce Data Warehouse Pipeline

An end-to-end ETL pipeline extracting data from **PostgreSQL** and loading it into **DuckDB** for analytical processing.

## Tech Stack
- **Source DB:** PostgreSQL
- **Data Warehouse:** DuckDB
- **Language:** Python (Conda environment)
- **OS:** Zorin OS (Linux)

## Project Structure
- `/scripts`: Python ETL scripts
- `/data`: Local storage for DuckDB files
- `requirements.txt`: Python dependencies
